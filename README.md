🚀 Maravilhas da Exploração Espacial

Este é um projeto web responsivo focado na divulgação de informações e documentários sobre o cosmos, destacando o Sistema Solar, a Missão Artemis e o Telescópio James Webb. 

O site foi desenvolvido do absoluto zero para consolidar meus aprendizados práticos em fundamentos de desenvolvimento web front-end.

## 🛠️ Tecnologias e Conceitos Utilizados

* **HTML5 Semântico:** Uso correto de tags como `<main>`, `<header>`, `<section>` e `<article>` para melhor acessibilidade e SEO.
* **CSS3 Avançado:** Criação de uma identidade visual temática (Dark Mode) utilizando degradês fixos (`linear-gradient` e `background-attachment: fixed`).
* **Box Model & Sombras:** Aplicação de espaçamentos (`padding`/`margin`) e profundidade visual com `box-shadow`.
* **Flexbox Layout:** Utilização de técnicas de alinhamento e distribuição de espaço no cabeçalho da página.
* **Animações e Interatividade:** Uso de pseudo-classes (`:hover`) combinadas com propriedades de transformação (`transform: rotate`) para criar efeitos dinâmicos ao interagir com os elementos.
* **Responsividade:** Layout adaptável e otimizado para visualização em dispositivos móveis (smartphones).
